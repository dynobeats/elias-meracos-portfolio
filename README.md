# Elias Meracos - portfolio

En statisk portfoliohemsida med CV, utbildningsprojekt och interaktiva demonstrationer inom nätverksteknik.

## Struktur

- `index.html` - startsida och felsökningsutmaning
- `cv.html` - publik CV-sida
- `projekt.html` - projektöversikt och kompetensmatris
- `projekt-aterbruk.html` - projekt om hårdvaruåterbruk
- `projekt-pfsense.html` - hybridlabb med pfSense
- `assets/` - publika dokument och presentationsbilder

## Publicering

Webbplatsen kräver inget byggsteg. I Cloudflare Pages används repositoryts rot som publiceringsmapp och inget build command anges.

## Integritet

Repositoryt ska endast innehålla den publika CV-versionen. Privat CV, temporära filer och lokala arbetsfiler ska inte publiceras.
